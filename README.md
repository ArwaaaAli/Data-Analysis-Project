# University Graduates Dataset Analysis – Data Analysis Project

##  Project Description
This project analyzes a dataset of university graduates in Saudi Arabia.  
The report includes screenshots of code snippets and detailed analysis of graduation trends.

**Key Features:**
- Technical description of the dataset (rows, columns, key variables)  
- Analysis using statistics, correlation, regression, PCA, and outlier detection  
- Insights on graduation trends by region, gender, major, and university
